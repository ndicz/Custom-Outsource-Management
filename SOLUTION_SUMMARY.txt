## 🔄 DATA FLOW DIAGRAMS

### PAYROLL WORKFLOW (Existing v18.0.2.0.0)
─────────────────────────────────────────
Employee Setup (PPh/BPJS)
    ↓
Salary Structure Definition
    ↓
Payroll Period (Jan/Feb/etc)
    ↓
Generate Payslip
    ├─ Auto-calculate PPh21 (progressive 5%-35% based on PTKP)
    ├─ Auto-calculate BPJS (JHT 5.7%, Kes 5%, JP 0.3%, JKK, JKM)
    ├─ Approval: Draft → Submit → Verify → Approve (multi-level)
    │
    ├─ GL Posting:
    │   Dr: 4101 Salary Expense    [Total Salary]
    │   Cr: 2101 Salary Payable    [Salary - Deductions]
    │   Cr: 2102 Tax Payable       [PPh21]
    │   Cr: 2103 BPJS Payable      [BPJS contribution]
    │
    └─ Bank Export & Payment
        └─ Export to: Mandiri / BCA / BRI / CSV

Monthly:
    └─ Tax Reporting:
        ├─ PPh21 Reconciliation
        ├─ e-Faktur Export (if applicable)
        └─ BPJS Submission (CSV format)


### PURCHASING & COSTING WORKFLOW (NEW)
──────────────────────────────────────
Create Purchase Order
    ├─ Link to Project = Client Outsource Contract
    ├─ Line Items:
    │   ├─ Material (quantity, project X, billable flag)
    │   ├─ Service (hours, project Y, billable flag)
    │   └─ Other (operational, project Z, billable flag)
    │
    └─ Auto-assign Analytic Account from Project
        ↓
        Cost Tracked to Client/Project
        ↓
        Used in:
        ├─ GL: Debit COGS / Debit Inventory
        ├─ P&L Report: Revenue - COGS = Gross Profit
        └─ Client Profitability: Per-client cost tracking


### OPERATIONAL EXPENSE WORKFLOW (NEW)
─────────────────────────────────────
Create Operational Expense
    ├─ Date / Type / Amount
    ├─ Attach Receipt/Bukti
    ├─ Optional: Link to Project (cost allocation)
    │
    └─ Workflow: Draft → Submit → Verify → Approve → Posted
                                              ↓
                           On APPROVE: Auto GL Posting
                           ├─ Dr: [Expense Account based on Type]
                           │   (e.g., 4201 Rent, 4202 Utilities)
                           ├─ Cr: [Bank/Cash Account] (1101)
                           └─ Analytic: [Project if linked]
                           ↓
                           GL Posting: POSTED
                           ↓
                           Visible in Financial Reports


### FINANCIAL REPORTING WORKFLOW (NEW)
──────────────────────────────────────
Generate Financial Report (P&L)
    ├─ Select Type: P&L / Balance Sheet / Cash Flow / Trial Balance
    ├─ Select Period: Start Date - End Date
    │
    └─ System Auto-Computes (from GL data):
        ├─ REVENUE
        │   ├─ Total from out_invoices (status='posted')
        │   └─ Breakdown per Customer/Client
        │
        ├─ COST OF GOODS SOLD
        │   ├─ Billable Purchases (cost_category = material/service)
        │   └─ Inventory Adjustments
        │
        ├─ GROSS PROFIT
        │   └─ Revenue - COGS
        │
        ├─ OPERATING EXPENSES
        │   ├─ Salary Expense (from payslips)
        │   ├─ Operational Expenses (12 types, 4201-4212)
        │   └─ Other Expenses
        │
        ├─ EBIT
        │   └─ Gross Profit - Total Expenses
        │
        ├─ TAX
        │   └─ EBIT × 21% (corporate tax rate, configurable)
        │
        └─ NET INCOME
            └─ EBIT - Tax
            ↓
            Display: HTML Format Table
            ├─ Summary Section
            ├─ Revenue Section per Client
            └─ Expense Breakdown


### CLIENT PROFITABILITY WORKFLOW (NEW)
────────────────────────────────────────
Generate Client Profitability Report
    ├─ Select Client (Partner)
    ├─ Optional: Select Project
    ├─ Select Period
    │
    └─ System Auto-Analyzes:
        ├─ REVENUE
        │   ├─ Total invoices for client (in period)
        │   └─ Invoice count
        │
        ├─ COSTS
        │   ├─ Material Costs (PO with cost_category='material')
        │   ├─ Service Costs (purchase.order.line with cost_category='service')
        │   └─ Allocated Overhead (10% of COGS, configurable)
        │
        ├─ PROFITABILITY
        │   ├─ Gross Profit = Revenue - Total Cost
        │   ├─ Gross Margin % = (Profit / Revenue) × 100
        │   ├─ Avg Invoice Value = Revenue / Invoice Count
        │   ├─ Cost per Invoice = Total Cost / Invoice Count
        │   └─ Profitability Ratio = Profit / Cost
        │
        ├─ PER-INVOICE BREAKDOWN
        │   ├─ Invoice #1: Revenue X, Cost Y, Margin Z%
        │   ├─ Invoice #2: Revenue X, Cost Y, Margin Z%
        │   └─ ... per invoice detail
        │
        └─ Display: HTML Analysis Table
            └─ Shows: Which clients most/least profitable?
                      Where to cut costs? Scale up?

## 🚀 HOW TO USE

### Installation:
1. Copy module to: C:\Program Files\Odoo 18.0.20260329\server\custom_addons\
2. Activate module: Settings → Apps → Install "Outsource Custom"
3. Create GL Accounts: 12 expense accounts + bank/cash account
4. Configure in Settings: Accounting → Outsource Custom Configuration
   → Map all 12 expense types to GL accounts

### Daily Use:

PAYROLL (Existing):
  • Create employees, set PPh/BPJS
  • Generate payslips monthly
  • Multi-level approval
  • Bank export → Payment

PURCHASING:
  • Create PO, link to project/client
  • Set cost category & billable flag
  • ✓ Cost auto-tracked to client

EXPENSES:
  • Create operational expense
  • Attach receipt
  • Workflow: Submit → Verify → Approve
  • ✓ GL auto-posts on approval

REPORTING:
  • Generate P&L reports (select period)
  • Analyze client profitability
  • Download/share HTML reports
